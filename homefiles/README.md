# File Description
## .bashrc file
This bashrc file allows the usage of alias, which provide a shortcut for using a certain commands.

## Symboliclink script
This script renames .bashrc file in the home directory into a backup file. Than it creates a symbolic link for the .bashrc located in the homefiles directory to the user home directory.
